# 📱 Проект: «Куда сходить в Оренбурге»

## 🎯 Цель проекта

Разработать современное мобильное приложение для просмотра мероприятий в
городе Оренбург с возможностью: - Просмотра афиши - Фильтрации по
категориям - Поиска мероприятий - Добавления в избранное - Покупки
билетов - Хранения электронных билетов - Управления профилем
пользователя

Аналог: Яндекс Афиша\
Стек: React Native + NestJS + Prisma + PostgreSQL

------------------------------------------------------------------------

# 📱 Экраны приложения

## 1️⃣ Главный экран --- Лента событий

### Содержит:

-   Заголовок: «Куда сходить в Оренбурге»
-   Подзаголовок
-   Строку поиска
-   Горизонтальные категории
-   Блок «Популярное»
-   Список карточек событий

### Карточка события:

-   Фото
-   Название
-   Краткое описание
-   Дата
-   Цена
-   Кнопка «Подробнее»
-   Иконка «Избранное»

### API:

-   GET /events
-   GET /events/popular
-   GET /categories
